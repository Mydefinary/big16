# ✅ 405 Method Not Allowed 문제 해결

## 원인 분석
**Frontend API 호출 경로 문제**:
- 이전: `baseURL: "/api"` + `/question-api/` = `/api/question-api/` ❌
- Gateway: `/question-api/**` → backend (라우팅 없음: `/api/**`)
- 결과: 405 Method Not Allowed

## 해결 방안
**Frontend 수정**:
1. `baseURL: "/api"` → `baseURL: ""` 변경
2. `/question-api/` → `/question-api/ask` 정확한 엔드포인트 지정

## 수정 결과
- **이전**: `/api/question-api/` → 405 Method Not Allowed
- **현재**: `/question-api/ask` → 400 Bad Request (엔드포인트 존재)
