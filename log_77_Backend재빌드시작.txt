# 방안 1 실행: Backend 소스코드로 재빌드

## 문제 상황
- 소스코드: `/question-api/ask` 엔드포인트 존재 (main.py:421)
- 배포된 이미지: `/question-api/ask` 404 Not Found
- 원인: 소스코드와 배포 버전 불일치

## 해결 방안
1. Dockerfile을 안전한 단일 스테이지로 변경
2. 원래 requirements.txt 사용 (의존성 충돌 방지)
3. hjlee7.azurecr.io/webtoon-question-backend-lee-2:latest로 빌드

## 현재 상태
- Backend 빌드 진행 중 (의존성 설치 단계)
- ragpool 노드에서 12Gi 메모리로 실행 예정
- 빌드 완료 후 push → 배포 → API 테스트 진행
